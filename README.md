# Sera_ME
🧠 Será M.E.? O "Será M.E.?" é um aplicativo web interativo, leve e intuitivo, desenvolvido para auxiliar profissionais de saúde na triagem rápida e identificação de casos suspeitos de Morte Encefálica (M.E.).  Baseado nas diretrizes e resoluções vigentes (como a Resolução CFM nº 2.173/17 no Brasil)
